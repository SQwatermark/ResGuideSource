<center><h1>调试快捷键</h1><p>Debug Keys</p></center>

## 调试快捷键丨Debug Keys

| 快捷键     | 作用                   |
| ---------- | ---------------------- |
| `F3` + `V` | 加载所有可见区块       |
| `F3` + `O` | 光影（着色器）设置     |
| `F3` + `R` | 重新加载光影（着色器） |

## 区块调试快捷键，见系统属性 "chunk.debug.keys"
| 快捷键         | 作用                   |
| -------------- | ---------------------- |
| F3 + E         | chunk path             |
| F3 + L         | smart cull             |
| F3 + U         | capture frustum        |
| Alt + F3 + U   | capture shadow frustum |
| Shift + F3 + U | release frustum        |
| F3 + V         | chunk visibility       |
