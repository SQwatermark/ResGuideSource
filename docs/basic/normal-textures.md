# 简单地修改一张纹理

打开 assets/minecraft/textures/block，找到要修改的纹理，这里我选择了 acacia_log.png。

::: tip

如果你实际看到的路径是 assets/minecraft/textures/blocks，别担心，Minecraft的不同版本的资源路径并不相同，用你实际看到的路径即可。

:::

打开图片编辑器，对其修改一番，并按照原名保存，覆盖掉原先的文件。

我仅仅改了下色调：
